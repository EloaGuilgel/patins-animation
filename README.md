# 🎨 CSS Animations

Neste projeto foram utilizadas **CSS Animations** para criar movimentos e efeitos visuais na interface da página.

A animação foi construída utilizando a regra:

```css
@keyframes
```

O **@keyframes** define as etapas de uma animação, descrevendo como um elemento deve mudar ao longo do tempo, permitindo criar transições suaves entre diferentes estados.

Exemplo:

```css
@keyframes slideIn {
  from {
    transform: translateX(-100px);
  }

  to {
    transform: translateX(0);
  }
}
```

Depois disso, a animação é aplicada ao elemento utilizando a propriedade:

```css
animation
```
